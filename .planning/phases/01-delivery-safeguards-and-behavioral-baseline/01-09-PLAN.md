<threat_model>
## Trust Boundaries
| Boundary | Description |
|---|---|
| Fork PR refs/paths/code → GitHub runner | Untrusted contributions control checked-out files and diff inputs. |
| CI jobs → artifacts/secrets/registry | Selected jobs may emit artifacts; publisher has registry write capability. |
| Fresh runner/package manifests → test processes | Clean machines must install only repository-declared Python/npm inputs before executing untrusted tests. |

## STRIDE Threat Register
| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|---|---|---|---|---|---|
| T-01-01 | Tampering | refs/path selector | high | mitigate | Full history, normalized relative paths, traversal rejection, argument arrays, tested matrix. |
| T-01-02 | Elevation | PR workflow | high | mitigate | `pull_request`, read-only defaults, no `pull_request_target`, no PR publish/login. |
| T-01-03 | Disclosure | artifacts | high | mitigate | Synthetic fixtures, failure-only scoped uploads, no env dumps, 14-day retention. |
| T-01-04 | Tampering | aggregate skips | high | mitigate | Always-present gate validates selected results and audited selection reasons. |
| T-01-05 | Tampering | generated report | high | mitigate | Always-on schema/discovery/byte-freshness validation. |
| T-01-06 | Spoofing | tag/image version | high | mitigate | Exact `v` + VERSION check and canonical build arg/OCI label. |
| T-01-07 | Elevation | publisher | high | mitigate | Package write permission exists only in reusable publisher downstream of gate. |
| T-01-08 | Tampering | runner dependency state | high | mitigate | Python 3.12 installs both checked-in requirement files; Node 20.19.x runs `npm ci --prefix frontend`; browser jobs use Playwright 1.61.1's pinned container and Chromium. |
| T-01-09 | Spoofing | Compose/CI build version | high | mitigate | Both paths consume validated `scripts/docker-version.mjs` output; contracts reject missing, malformed, conflicting, or hardcoded input. |
| T-01-SC | Tampering | dependencies/actions | high | mitigate | Exact npm lock/pinned Chromium and major-pinned official Actions from audited workflow. |
</threat_model>
